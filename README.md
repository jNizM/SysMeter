# SysMeter by jNizM
Shows ComputerInfo via AutoHotkey

  
## Features
* Shows Date, Time & Uptime (Toggle)
* Shows CPU Usage
* Shows RAM & SWAP Usage
* Shows Drives Usage
* Toggle MB/GB to Percentage
* Toggle & Reset Transparency
* Toggle AlwaysOnTop
* Color Scheme
* Save Ini-Settings
* Reload Gui on Device Changes
